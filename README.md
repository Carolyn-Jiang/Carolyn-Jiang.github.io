# Carolyn Jiang — Personal Academic Homepage

A clean, minimal personal academic website built with plain HTML and CSS.

## Structure

```
.
├── index.html          # Home (profile, name, position, research interests)
├── publications.html   # Publications
├── service.html        # Academic Service
├── style.css           # Shared styles
├── assets/
│   └── profile.jpg     # Profile photo
└── README.md
```

## Deployment (GitHub Pages)

This site is deployed with GitHub Pages from a user repository
(`Carolyn-Jiang.github.io`):

1. Push the files to the `main` branch of the repository.
2. In the repository, go to **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to *Deploy from a branch*.
4. Select the **main** branch and the **/ (root)** folder, then **Save**.
5. The site will be published at: <https://Carolyn-Jiang.github.io>

Because the repository is named `Carolyn-Jiang.github.io`, GitHub Pages serves
`index.html` at the root URL automatically. No build step or framework is
required.
